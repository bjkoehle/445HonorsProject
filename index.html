Google Web Service

This project was developed as an addition the the CSE 445 class at ASU for honors credit. The requirements of the project are:

  • To write a SOAP service for accessing the google api to get the distance and drive time between 2 cities, using the service at: https://developers.google.com/maps/documentation/distancematrix/.
  • Host this service and write a web application that can consume and run tests on the service.
  • Additionally create and consume one more service that is based off the api located at: https://developers.google.com/maps/documentation/webservices/.
